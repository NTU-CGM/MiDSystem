Introduction

MiDSystem is a comprehensive online system for de novo assembly and analysis of microbial genomes. It supports both whole genome assembly (Figure 1) and metagenomics analysis pipelines (Figure 2) in shotgun sequencing data.

Figure 1 The workflow of the de novo assembly.

Figure 2 The workflow of metagenomics analysis.


Getting started

To get started, select Reference-Guided Assembly, Non-Refernce Assembly, or Long-read/Hybrid Assembly for a single bacterial genome, click Metagenomics Analysis for shotgun metagenomic sequencing. See Help for the tutorial on how to use MiDSystem and links to demonstration reports, About for the system information.
